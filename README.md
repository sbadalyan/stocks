# Stocks API visualization 

## Install dependencies

```sh
npm i
```

## Create env file

```sh
cp .example.env .env
```

Add your Alpha Vantage API key to the API_KEY variable in the .env file.

## Start iOS app

```sh
cd ios && pod install && cd ..
```

```sh
npm run ios
```

## Start Android app

```sh
npm run android
```
